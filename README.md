# Retail360

## Enterprise Retail Intelligence Platform

Retail360 is an end-to-end enterprise retail analytics platform that collects data from multiple sources, processes it through automated ETL pipelines, stores it in a cloud data warehouse, and delivers business insights through interactive dashboards.

## Tech Stack

- Python
- PostgreSQL
- Snowflake
- dbt
- Apache Airflow
- Power BI
- REST APIs
- Azure Blob Storage
- Git & GitHub
